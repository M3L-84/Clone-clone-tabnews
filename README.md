# Clone-clone-tabnews

Aprendendo a programar 1% - Review
Implementação do https://www.tabnews.com.br para o https://curso.dev

1. Criar um na aba Explorador um arquivo .nvmrc
   - Ao abrir o arquivo inclua no .nvmrc :
     lts/hydrogen e 'enter' para adicionar mais uma linha.

2. No terminal digite:
   - npm init

   Em seguida aperte 'enter' até 'license: (ISC)' e digite MIT
   Após isso, aperte 'enter' até o final.
   - npm install next@13.1.6

   - npm install react@18.2.0

   - npm install react-dom@18.2.0

3. Click em package.json , abaixo do "scripts" há uma atalho de nome "teste", apague-o e digite:
   "dev": "next dev" e salve (crtl + s)

4. No terminal digite:
   - git log
     resposta: Initial commit

   - git log --stat (mostra de forma resumida o que foi criado no github)

5. GITHUB
   - Crie o arquivo .gitignore
   - git add .gitignore
   - git status
   - git commit -m "frase ou texto"
   - git commit --amend (imenda o arquivo)
   - git status (para conferir o que será enviado)
   - git push (para empurrar o para o repositório do github)
